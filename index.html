Multiple Elements Cycle Plugin

Provides a simple preview scrolling panel of list items. Shows the given range of li items from the middle (hiding the edges items) and allows scrolling left and right within the list. Does not handle automatic scrolling or wrapping items around.

This script was designed to be lightweight. If you are looking for more advanced script then I recommend the jQuery Cycle Plugin

It styles list items which can contain images, videos, text or combinations.

Installing

Step 1: Include the files in your Project

You need to include the jQuery library. This comes bundled with jQuery 1.3.2. You also need to include the jquery.multipleelements.cycle.js in your page. 

<script type="text/javascript" src="jquery-1.3.2.min.js"></script>
<script type="text/javascript" src="jquery.multipleelements.cycle.js"></script>

Step 2: Setup your HTML

<div class="cycleElementsContainer" id="plainDemo">
	<a href="#" id="cycleElementsLeft">Scroll Left</a>
	<div id="cycleElements">
		<ul>			
			<li>#1</li>
			<li>#2</li>
			<li>#3</li>
			<li>#4</li>
			<li>#5</li>	
			<li>#6</li>
		</ul>
	</div>
	<a href="#" id="cycleElementsRight">Scroll Right</a>
</div>

Step 3: Configure

Once you have included the JS and the html then you need to define your own CSS styles and initialize the JS effect

/* CSS */
.cycleElementsContainer li { width: XXXX; float: left; }

/* HTML */
<script type="text/javascript">
	$(document).ready(function() {
		$("#plainDemo").multipleElementsCycle();
	});
</script>

Configuration

$(document).ready(function() {
	$("#plainDemo").multipleElementsCycle({
		elementContainer: '#cycleElements',	// Selector for element (ul) container (Selector)
		prevElement: '#cycleElementsLeft',	// Selector to scroll previous (Selector)
		nextElement: '#cycleElementsRight',	// Selector to scroll next (Selector)
		speed: 500,				// Speed to scroll elements (Int)
		containerWidth: false,			// Override default width (Int)
		showCount: 4				// Items to show from the list (Int)
		overrideStart: false			// Override the start with a defined value (int)
	});
});
